Project WebXP

Run Windows XP in your browser and more!

About

Project WebXP allows developers and nostalgic users to run Windows XP in their browser. Future plans include support for Windows Vista, Windows 8/8.1, Windows 10, and Ubuntu.

Features

  • Windows XP in the browser with a few games installed
  • Future support for Windows Vista
  • Future support for Windows 8/8.1
  • Future support for Windows 10
  • Future support for Ubuntu

System Requirements

To run games smoothly, you should have at least 8GB of RAM. Otherwise, any modern system should be able to run it.

Support

If you encounter any issues, please submit an issue on our GitHub repository.
